# Ultimate Drink Guide

Webapp created to show how to prepare different drinks and view their ingredients found in [TheCocktailDB](https://www.thecocktaildb.com/api.php) API.

## Live Demo
https://splendid-creponne-acaaa4.netlify.app

## Setup
### Pre requisites
- Node.js v16.10.0 or newer

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

## Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
